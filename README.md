# ⚙️ Proyectos

# 🚗 Análisis del Mercado de Vehículos Usados
*Desarrollo de una aplicación web interactiva para la exploración de datos automotrices.*

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://proyectosprint7-gjfi.onrender.com)

## 📖 Resumen del Proyecto
Este proyecto consiste en una herramienta de Análisis Exploratorio de Datos (EDA) construida para investigar un conjunto de datos masivo de anuncios de venta de vehículos en Estados Unidos. A través de una interfaz interactiva, los usuarios pueden analizar cómo variables como el kilometraje, el año de fabricación, el tipo de transmisión y la condición del vehículo impactan en su precio final de venta.

## 🎯 Objetivos Clave
1. **Preprocesamiento y Limpieza de Datos:** Identificar y tratar valores ausentes, corregir tipos de datos y estandarizar registros para asegurar la calidad del análisis.
2. **Análisis Exploratorio (EDA):** Descubrir patrones, tendencias y valores atípicos (outliers) en el mercado de autos usados.
3. **Despliegue Interactivo:** Construir y desplegar un panel de control (dashboard) web accesible para usuarios no técnicos.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python
- **Manipulación de Datos:** Pandas
- **Visualización:** Plotly, Matplotlib
- **Desarrollo Web:** Streamlit
- **Entorno:** Entornos virtuales con Anaconda

## 🔍 Hallazgos Principales
- **Depreciación por Kilometraje:** Se observó una correlación negativa fuerte entre el precio y el odómetro; los vehículos superando las 100,000 millas muestran una caída drástica en su valor.
- **Preferencias de Mercado:** Las carrocerías tipo SUV y Sedán dominan el volumen de anuncios, pero las categorías de camionetas (trucks) mantienen un valor residual más alto.
- **Impacto de la Condición:** La diferencia de precio promedio entre un vehículo en estado "excelente" frente a uno "bueno" es del 32.85%.
